Partition Function mod l

About

In number theory, a/an (integer) partition of a positive integer n is a way of writing n as a sum of positive integers.

The partition function p(n) represents the number of possible partitions of a natural number n.

We studied the distribution of p(n) mod l for l = 2, 3, 4, 5, 6.
